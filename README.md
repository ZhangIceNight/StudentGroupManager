# 学生分组项目

这个项目用于根据学生名单生成随机分组。

## 安装

1. 克隆仓库：
   ```bash
   git clone https://github.com/ZhangIceNight/StudentGroupManager.git
   ```
2. 进入项目目录：
   ```bash
   cd YOUR_CODE_PATH
   ```
3. 项目依赖：
   ```bash
   pip install pandas
   ``` 

## 使用

运行以下命令生成学生分组：
```bash
python main.py
```
生成的分组会输出到 CSV 文件中。

## Tip
如果输入是xls等格式，有时会出现编码错误，这时创建一个新的空csv文件，然后手动粘贴过来就可以，另存为是不可以的
